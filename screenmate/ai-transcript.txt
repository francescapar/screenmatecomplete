Step 3 – Initialize Git Repository

Run the following commands in the project folder:

git init
git add .
git commit -m "Initial commit: setup project structure"

--------------------------------------

Step 4 – Connect GitHub Repository

git remote add origin https://github.com/YOURUSERNAME/screenmate.git
git branch -M main
git push -u origin main

--------------------------------------

Step 5 – README File

README.md

# ScreenMate

## Project Description
ScreenMate is a web application that helps users track movies and TV shows they want to watch. Users can create a watchlist, mark titles as watched, and organize what they plan to watch next.

## Planned Features
- Add movies or TV shows to a watchlist
- Mark titles as watched
- Filter watchlist by watched/unwatched
- Suggest a “Next to Watch” movie
- Search for movies by title

## Tech Stack
HTML
CSS
JavaScript
